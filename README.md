.emacs.d
========

## Installing

```
git clone --recurse-submodules -j8 https://github.com/themattchan/emacs.d.git ~/.emacs.d
cd ~/.emacs.d && make init
```

`git clone --recursive` on old versions of git


## Updating

```
make update
```

## Compiling

```
make compile
```

to remove compiled shit

```
make clean
```
